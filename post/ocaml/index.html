Getting Setup with Ocaml

31 March 2017

PreRequisite

This post assumes you have spacemacs already installed.

Install Ocaml

Follow the installation instructions from RealWorldOcaml till you hit the Editors section

After this step you should have a working installation of * ocaml * opam ( the package manager) * merlin * ocamllint * utop

The contents of ~/.ocamllint should be very close to:

## Setting up spacemacs
(* Added by OPAM. *)
let () =
  try Topdirs.dir_directory (Sys.getenv "OCAML_TOPLEVEL_PATH")
  with Not_found -> ()
;;
(* Manually Added*)
#use "topfind";;
#thread;;
#camlp4o;;
#require "core.top";;
#require "core.syntax";;

Project setup

Ensure there is a .merlin file in the same directory as your hello.ml file.

The contents of the .merlin file should be:

PKG core lwt

This doc explains why that statement is needed.

That’s it! now you can open spacemacs and enjoy a full featured IDE experience for ocaml.

ocaml setup merlin ocamllint spacemacs
